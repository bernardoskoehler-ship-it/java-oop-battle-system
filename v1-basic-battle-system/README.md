# RPG — Version 1 (Basic Battle System)

## Description

This is the first version of a Java RPG system.

It focuses on building a basic battle system using OOP concepts.

## Features

- Abstract class (Personagem)
- Inheritance (Guerreiro, Mago, Arqueiro)
- Polymorphism
- Turn-based combat
- Defense system
- Resource management (mana, arrows)

## Concepts Practiced

- Encapsulation
- Inheritance
- Polymorphism
- Clean code structure
