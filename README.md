# Java OOP RPG

This project is a simple RPG-style battle system built in Java to practice Object-Oriented Programming (OOP).

## 🚀 Concepts Used

* Abstract Classes
* Inheritance
* Polymorphism
* Encapsulation (basic)
* Method overriding

## ⚔️ Characters

The system includes different types of characters:

* **Warrior** → high damage, stronger when low HP
* **Mage** → deals damage but takes recoil
* **Archer** → chance-based critical attacks

Each character has its own behavior implemented using polymorphism.

## 🧠 How It Works

Characters are stored in a `Personagem[]` array and interact with each other through their `executarAcao()` method.

This demonstrates how different objects can share the same interface but behave differently.
