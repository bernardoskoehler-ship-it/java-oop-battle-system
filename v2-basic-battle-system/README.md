# ⚔️ V2 - Basic Battle System (Java)

A simple turn-based battle system developed in Java as part of my learning journey in Object-Oriented Programming.

---

## 🚀 Overview

This project simulates a duel between characters with different classes and abilities.
Each character has its own behavior, resources, and combat logic.

---

## 🧠 Concepts Practiced

* Object-Oriented Programming (OOP)
* Encapsulation
* Inheritance
* Polymorphism
* Abstraction
* Interfaces
* Code organization and refactoring

---

---

## ⚔️ Features

* Turn-based combat system
* Multiple character classes:

  * 🛡️ Guerreiro (defense system)
  * 🔥 Mago (mana-based abilities)
  * 🏹 Arqueiro (arrow-based abilities)
* Critical hit system
* Resource management (mana / arrows)
* Defensive mechanics
* Automatic action decision (AI behavior)

---

## ▶️ How to Run

1. Open the project in IntelliJ IDEA (or any Java IDE)
2. Run the `Main.java` file
